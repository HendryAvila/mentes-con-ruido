# Mentes con Ruido — Productividad para Mentes Creativas

> Curso interactivo basado en neurociencia, ciencia de habitos y tecnicas de aprendizaje modernas. Para quienes ya probaron todo y nada les duro.

## Sobre este proyecto

Este curso interactivo fue **orquestado por [Hendry Avila](https://github.com/hendryavila)** y **generado con asistencia de Inteligencia Artificial**.

### Filosofia

Este proyecto es **Open Source** y su unico proposito es **la educacion**. Nace de la conviccion de que el aprendizaje debe ser:

- **Accesible**: Gratis, abierto, y disponible para todos
- **Dinamico**: Interactivo, no solo texto estatico
- **Adaptable**: Cada persona aprende a su ritmo
- **Practico**: Aprender haciendo, no solo leyendo

### Nota sobre la implementacion tecnica

El desarrollo tecnico de este curso (codigo, estructura, patrones) puede o no seguir las mejores practicas de ingenieria de software. **Ese NO es el foco del proyecto.** El objetivo es proveer un espacio de aprendizaje diferente, dinamico y efectivo. Si encuentras mejoras tecnicas, los PRs son bienvenidos.

## Que vas a aprender

- Por que tu atencion en pantalla dura solo 47 segundos (Gloria Mark, 20 anios de datos)
- Por que las mentes creativas pierden el foco — y por que eso es un superpoder, no un defecto
- Como construir habitos que duren usando ciencia real (no tips de Instagram)
- Como entrar en estado de flow intencionalmente
- Tecnicas de aprendizaje que REALMENTE funcionan (y cuales no, segun Dunlosky 2013)
- Como disenar tu entorno para que la productividad sea el camino de menor resistencia
- Como usar IA sin perder la capacidad de pensar por vos mismo

## Tech Stack

- [SvelteKit 5](https://svelte.dev/) — Framework web
- [Tailwind CSS v4](https://tailwindcss.com/) — Estilos
- [Static Adapter](https://kit.svelte.dev/docs/adapter-static) — Deploy estatico
- GitHub Pages — Hosting

## Como ejecutar localmente

```bash
git clone https://github.com/hendryavila/mentes-con-ruido.git
cd mentes-con-ruido
npm install
npm run dev
```

Abre `http://localhost:5173` en tu navegador.
